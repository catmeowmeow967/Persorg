# TaskBoard (Система управления задачами)

Профессиональное desktop-приложение для управления проектами и задачами в стиле Kanban. Создано на Java с современным графическим интерфейсом. Позволяет пользователям организовывать рабочие процессы, отслеживать сроки выполнения задач, управлять приоритетами и совместно работать над проектами.

---

## О проекте

### Почему был создан этот продукт?

TaskBoard появился как практическая реализация цифровой системы управления задачами в формате локального desktop-приложения. Проект объединяет потребности пользователей в планировании задач, контроле сроков и организации рабочих процессов с инструментами для управления проектами, колонками и участниками. Разработка велась как комплексный пример полноценного корпоративного приложения с архитектурой MVC, реляционной базой данных и многооконным интерфейсом.

### Какую проблему он решает?

Приложение предоставляет единую цифровую среду для организации рабочих процессов без зависимости от облачных сервисов. Оно автоматизирует рутинные операции: создание задач, управление колонками, отслеживание сроков, назначение приоритетов, архивация выполненных задач, заменяя разрозненные заметки и таблицы структурированной Kanban-системой.

### В чем его ключевые преимущества?

**Полная локальность и безопасность:** Все данные хранятся в локальной базе MySQL на вашем компьютере

**Управление проектами:** Создание и переключение между несколькими рабочими досками

**Kanban-доска:** Визуальное представление задач в колонках с возможностью управления статусами

**Приоритеты и сроки:** Назначение уровней важности и дедлайнов для каждой задачи

**Система меток:** Категоризация задач цветовыми метками для быстрой идентификации

**Архивация задач:** Возможность архивировать выполненные задачи с последующим восстановлением

**Управление участниками:** Добавление пользователей в проекты с разграничением ролей

**Автоматизация:** Настройка правил для автоматических действий при изменении задач

**Фильтрация и поиск:** Быстрый поиск и фильтрация задач по приоритету, сроку и статусу

**Готовое рабочее решение:** Полнофункциональное приложение с типовыми сценариями работы из коробки

## Технологии

Для работы приложения требуется:

**Java Runtime Environment (JRE) 17 или выше:** [Скачать с официального сайта](https://www.oracle.com/java/technologies/downloads/)

**СУБД MySQL 8.0 или выше:** [Скачать установщик](https://dev.mysql.com/downloads/mysql/)

### Архитектура проекта:

| Компонент | Технология |
|-----------|------------|
| Язык и интерфейс | Java 17, JavaFX для современного GUI |
| База данных | MySQL 8.0 с полной схемой пользователей, проектов, колонок, задач и меток |
| Архитектура | MVC (Model-View-Controller) с четким разделением ответственности |
| Дополнительные компоненты | DAO-паттерн для доступа к данным, шифрование паролей (SHA-256), защита от SQL-инъекций (PreparedStatement) |

## Быстрая установка и запуск

Вы можете запустить приложение за несколько минут, используя готовые файлы:

### 1. Установите необходимое ПО

Установите Java (JRE) по ссылке выше.

Установите MySQL Server. Во время установки запишите заданный вами пароль для пользователя root.

### 2. Настройте базу данных

Скачайте готовый дамп базы данных.

Откройте командную строку MySQL (или общую командную строку/терминал).

Выполните команду для импорта данных, указав правильный путь к скачанному файлу. Вас попросят ввести пароль пользователя root MySQL.

### 3. Запустите приложение

Скачайте приложение из репозитория.

Запустите его двойным кликом или из командной строки.

**Важно:** Перед запуском убедитесь, что сервис MySQL Server запущен в вашей системе.

## Возможности

TaskBoard предоставляет полный цикл управления задачами:

**Система пользователей:** Регистрация и авторизация с хранением данных в локальной базе

**Управление проектами:** Создание, редактирование и удаление рабочих досок с цветовой индикацией

**Kanban-колонки:** Создание произвольного количества колонок для статусов задач

**Управление задачами:** Создание карточек задач с названием, описанием, приоритетом и сроком

**Приоритеты:** Четыре уровня важности: низкий, средний, высокий, критический

**Сроки выполнения:** Установка дедлайнов с цветовой индикацией просроченных задач

**Система меток:** Создание цветовых меток для категоризации задач

**Архивация:** Перемещение задач в архив с возможностью восстановления или полного удаления

**Управление участниками:** Добавление пользователей в проекты с ролями

**Автоматизация:** Настройка правил для автоматических действий при изменении задач

**Фильтры и поиск:** Поиск по тексту, фильтрация по приоритету, сроку и статусу

**Уведомления:** Просмотр задач на сегодня

**Вложения:** Прикрепление файлов к задачам

## Использование

### Начало работы

При первом запуске зарегистрируйтесь или войдите под существующими данными.

### Главный экран

После входа откроется рабочее пространство с Kanban-доской текущего проекта.

### Основные функции (панель инструментов)

| Раздел | Описание |
|--------|----------|
| **Доски** | Создание и переключение между проектами |
| **Фильтры** | Поиск и фильтрация задач по параметрам |
| **Метки** | Управление цветовыми метками для задач |
| **Участники** | Добавление и управление пользователями проекта |
| **Автоматизация** | Настройка правил автоматических действий |
| **Архив** | Просмотр и восстановление архивированных задач |
| **Сегодня** | Просмотр задач с дедлайном на текущий день |

### Работа с данными

Двойной клик по карточке задачи открывает детальное редактирование

Все изменения сохраняются автоматически через транзакции БД

Архивация задач через меню карточки

## Разработка и тестирование

Проект построен на архитектуре MVC с четким разделением:

**Модели:** User, Project, Column, Task, Label, Attachment, ProjectMember, AutomationRule и др.

**Контроллеры:** MainController, LoginController, ColumnController, TaskCardController, TaskDetailController, FiltersController, LabelsController, MembersController, AutomationController, ProjectSelectorController и другие для каждого окна

**Представления:** FXML-файлы с современным UI

**DAO-слой:** Отдельные классы для работы с каждой сущностью в БД

**TaskBoard** — это не просто список задач, а полноценная система управления проектами с акцентом на локальность данных, визуальную организацию рабочих процессов и инструменты совместной работы.
